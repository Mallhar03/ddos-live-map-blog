1. Random IP Generator

std::string getRandomIP(std::mt19937 &gen) {
    std::uniform_int_distribution<> dist(0, 255);
    return std::to_string(dist(gen)) + "." +
           std::to_string(dist(gen)) + "." +
           std::to_string(dist(gen)) + "." +
           std::to_string(dist(gen));
}

Insight: Each dist(gen) call pulls a fresh number from the generator, ensuring all 4 octets of the IP look random. The key ‚Äúaha‚Äù moment was realizing how distribution maps the random sequence to the valid IP range.

2. Timestamp Generator

std::string getTimestamp() {
    auto t = std::time(nullptr);
    auto tm = *std::localtime(&t);
    std::ostringstream oss;
    oss << std::put_time(&tm, "%Y-%m-%d %H:%M:%S");
    return oss.str();
}

Insight: The chain goes: epoch time ‚Üí local struct ‚Üí formatted string. Using oss (scratchpad) + put_time made it clear how raw time data turns into readable logs.
