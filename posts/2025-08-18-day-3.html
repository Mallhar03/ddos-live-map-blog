Project DDoS Visualiser – Day 3: From Packets to Logs

🛡️ Project Aim Recap

We’re building a C++ powered DDoS Attack Visualiser that can generate, read, and later map simulated attack logs into meaningful visuals.

🌍 Where We Left Off

  • Day 1: Understood what DDoS is, and how botnets + C2 work.
  • Day 2: Broke down attack types (Volumetric, Protocol, Application-layer).
  • Day 3: Moved from theory → implementation by writing our mock DDoS log generator in C++.

⚙️ The Deliverable

  • ✅ Generate fake log entries
  • ✅ Save them to ddos_log.txt
  • ✅ Push the working code on GitHub

🧩 Breaking Down the Functions

This is how I understood them piece by piece:

  • getRandomIP() → creates IPv4 addresses using random distribution.
  • getTimestamp() → grabs current time (epoch → local → formatted).
  • getRandomAttackType() → picks one attack type from a vector.
  • generateLogEntry() → assembles log line: timestamp | ip | attack_type.
  • main() → loops and writes entries into ddos_log.txt.

🐞 Debugging Story

Got hit with undefined reference errors — caused by tiny typos (getTimestamps vs getTimestamp).

Lesson: Declarations, definitions, and calls must match exactly in C++.
